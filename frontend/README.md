# Race Stats Frontend

Simple React frontend for displaying race results using Feature-Sliced Design architecture.

## Structure

```
src/
  app/          # App initialization
  pages/        # Page components
  features/     # Feature components (race-results)
  shared/       # Shared utilities, types, API
```

## Setup

```bash
yarn install
```

## Development

```bash
yarn dev
```

The app will run on http://localhost:5173

Make sure the backend API is running on http://localhost:3000

## Build

```bash
yarn build
```
